### 📍 Evidence Location #1: Raw HTTP Request (The Ground Truth)

**File:** `output/evidence/{timestamp}/requests/{test_id}_req.json`

The injection point is **visibly embedded** in the request structure:

```json
{
  "test_case_id": "A03-SOQL-001",
  "injection_point": {
    "location": "query_parameter",
    "field_name": "q",
    "original_value": "SELECT Id FROM Account",
    "injected_value": "SELECT Id FROM Account WHERE Name LIKE '' OR '1'='1",
    "mutation_type": "soql_injection"
  },
  "request": {
    "method": "GET",
    "url": "https://your-sandbox.my.salesforce.com/services/data/v58.0/query/?q=SELECT+Id+FROM+Account+WHERE+Name+LIKE+%27%27+OR+%271%27%3D%271",
    "headers": { ... },
    "body": null
  }
}
```

**Injection Point Types Captured:**
| `location` Value | Meaning | Example |
|-----------------|---------|---------|
| `query_parameter` | URL query string (`?q=...`) | `GET /search?q=<payload>` |
| `url_path` | URL path segment | `GET /users/<payload>/profile` |
| `json_body` | JSON request body | `POST /api { "name": "<payload>" }` |
| `form_body` | Form-encoded body | `POST /login username=<payload>` |
| `header` | HTTP header | `X-Custom-Header: <payload>` |
| `cookie` | Cookie value | `session_id=<payload>` |
| `multipart` | File upload field | `POST /upload file=<payload>` |

### ✅ Complete Injection Point Tracking Summary

| Evidence Type | Where Injection Point Is Documented |
|--------------|-------------------------------------|
| **Raw JSON Evidence** | `injection_point.location`, `injection_point.field_name`, `injection_point.json_path` |
| **HTML Report** | Prominent badge at top of finding card + vulnerable code section |
| **Proxy Logs** | Visible in URL, headers, or body + `X-SecTest-Parameter` header |
| **Test Plan** | `injection_target.type`, `injection_target.field`, `injection_target.selector` |
| **Context Analysis** | `parameters[].location`, `parameters[].name`, `parameters[].risk_assessment` |
| **Mutation Log** | `mutation_applied.target_location`, `mutation_applied.target_field`, `injection_offset` |
| **cURL Command** | Visible in the actual command string |

### 🎯 The Bottom Line

**Every single finding includes:**
1. ✅ **Exact Location Type** (query param, JSON body, URL path, header, cookie)
2. ✅ **Field Name** (`q`, `IsDeleted`, `recordId`, `X-Custom-Header`)
3. ✅ **JSON Path** (if applicable: `$.user.name`, `$.IsDeleted`)
